# Oasis Maternity and Nursing Home Mosobeti

A complete, single‑page website for Oasis Maternity and Nursing Home, a healthcare facility in Mosobeti, Kenya.  
It includes patient registration, online appointment booking, an admin dashboard, simulated SMS & M‑PESA, health tips, gallery, and more – all using **localStorage** for data persistence.

## 🏥 Features

- **Public Website** – Home, About, Services, Team, Departments, Facilities, Health Tips, Testimonials, Gallery, Insurance, FAQ, Emergency, Careers, Legal, Contact.
- **Patient Registration** – Create an account with personal and next‑of‑kin details.
- **Appointment Booking** – Choose a service, date, and receive confirmation.
- **Admin Dashboard** – View/manage appointments and patients (password: `admin123`).
- **Simulated SMS** – Toast notifications mimic SMS confirmations.
- **M‑PESA Payment** – Demo payment button (integration ready).
- **Fully Responsive** – Works on all screen sizes.
